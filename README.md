# BotaWorld — Эволюция Боты

Прототип мобильной игры для детей на SwiftUI (iOS 16+).

## Описание
Игра в жанре virtual pet + приключение. Игрок растит, эволюционирует и путешествует с персонажем Бота (детёныш верблюда) по карте Казахстана.

## Реализованные функции
- Вылупление Боты + ввод имени ребёнка
- Система эволюции (3 этапа)
- Карта Казахстана с регионами (Алматы, Астана, Степь)
- 3 мини-игры (Математика, Память, Быстрые нажатия)
- Ежедневный сундук с наградами
- Родительский режим с PIN
- Сохранение прогресса (UserDefaults)
- Ночной/дневной режим

## Как запустить
1. Открой проект в Xcode 14+
2. Выбери устройство или симулятор
3. Нажми **Run**
